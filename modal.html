What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page in a tree-like structure that allows programs (like JavaScript) to interact with, modify, and manage the content, structure, and style of a document dynamically.

How do you select an element from the DOM?

Use getElementById() for single, unique id selections. Use getElementsByClassName() or querySelectorAll() for selecting multiple elements with the same class. Use querySelector() and querySelectorAll() for flexible, CSS-like selectors.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to handle events at a higher level in the DOM tree, rather than attaching event listeners to individual elements. By using event delegation, you can manage events for multiple child elements by attaching a single event listener to a parent element, taking advantage of the event bubbling mechanism.
